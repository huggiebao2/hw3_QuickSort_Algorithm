# hw3_QuickSort_Algorithm 作業說明

## 說明

Quick Sort 演算法的部分，放在 `quick_sort()` 函式內。

作業要求要運行的範例整數陣列則是放在 `main()` 中，透過在 `main()` 中呼叫 `quick_sort()` 來為範例整數陣列進行排序。
